## Usage

除了 `gcc` 和 `python3` 手动安装，其他工具都是脚本自动安装。
其中 `python3` 在 ubuntu 上默认安装为好了，还需要额外安装一些包。

``` sh
sudo apt install -y python3-distutils
sudo apt install -y python3-setuptools
```
